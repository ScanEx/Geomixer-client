Тип данных Geometry

Пример точечной геометрии (на карте) 

{ 
  type: "POINT", 
  coordinates: [37.0, 55.0] 
}

Пример линейной геометрии (на карте) 

{ 
  type: "LINESTRING", 
  coordinates: [[32.3437,60.2308],[38.1445,58.6189],[37.6171,55.6581],[49.0429,46.1433]]
}

Пример полигональной геометрии (на карте) 

{ 
  type: "POLYGON", 
  coordinates: [[[29.8828,59.4345],[26.5429,51.0357],[44.1210,44.5318],[57.3046,54.3447],[46.7578,61.8501],[29.8828,59.4345]]]
}
Примечание: последняя точка полигона должна совпадать с первой.

}

Тип данных Style

Пример стиля для точечных объектов (на карте) 

{
  marker: {
    image: "http://maps.kosmosnimki.ru/api/img/marker_flag.png",
    dx: -2,
    dy: -20
  }
}

Пример стиля для линейных объектов (на карте) 

{
  outline: {
    color: 0x0000ff,
    thickness: 1,
    opacity: 100
  }
}

Пример стиля для пунктирной линии (на карте) 

{
  outline: {
    color: 0x000000,
    thickness: 1,
    opacity: 100,
    dashes: [3,2,1,2]
  }
}

Пример стиля для полигональных объектов (на карте) 

{
  outline: {
    color: 0x0000ff,
    thickness: 1,
    opacity: 100
  },
  fill: {
    color: 0xff0000,
    opacity: 50
  }
}

Пример стиля для полигона с паттерном (на карте) 

{
  outline: {
    color: 0xff0000,
    thickness: 1,
    opacity: 50
  },
  fill: {
    image: "http://kosmosnimki.ru/geomixer/docs/api_samples/car_icon_small.gif"
  }
}

Пример стиля для объекта c подписью (на карте) 

{
  label: {
    color: 0xff00ff,
    size: 12,
    haloColor: 0xffffff
  }
}
