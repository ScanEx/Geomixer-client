Upload file
Allows to upload small shp/tab/kml file on the fly in the form of user objects. After uploading user can edit geometry of objects and download objects in the same format kit.
Link to the map
Link saves current map state, and also user objects and texts.
Embed code
Embedding the map: the user can embed a map window into their site by copying and pasting an HTML snippet. An API key for the site's domain is required.
Print
Print current map fragment.
