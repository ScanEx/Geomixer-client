GeoMixer {gmxVersion} ({gmxData}) - this web application for interactive geodata view and access to basic data sources in the Internet, such as: Kosmosnimki.ru and OpenStreetMap

Using GeoMixer you can publish own geodata in intranet or in the Internet, put them over basic data coverages and give access to them for however large number of users, changing access permissions.

GeoMixer also includes API component, which enables integrating the created projects into other web sites and applications and carry out program control over interactive map functional.
