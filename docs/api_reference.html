Окно карты
Конструктор
Класс Параметры Описание
makeFlashMap (HTML element, map ID) Базовый конструктор: создает flash-объект карты в указанном HTML-контейнере. Карта загружается по заданному ID или названию (например, "Kosmosnimki")
Метод Параметры Описание
map.moveTo (X, Y, Z) При загрузке карта позиционируется по указанным координатам (X - начальная координата по долготе, Y - начальная координата по широте) и на заданном уровне детализации (Z)
map.moveToCoordinates (X, Y) В отличие от map.moveTo возможно задавать координаты в любом формате, например ("61.3266 N, 45.0878 E")

Конструктор
Класс Параметры Описание
createKosmosnimkiMap (HTML element, host name, map ID) Базовый конструктор: создает flash-объект карты с указанным ID поверх базовой карты "Kosmosnimki". Добавляет в интерфейс карты переключатель базовых слоев "Карта-Снимки"
Метод Параметры Описание
map.setMode ("satellite", "map") Задает базовый слой, видимый по умолчанию: "map" - базовая растровая карта
"satellite" - базовое спутникове покрытие
Слои
Fuction(Map)
Класс Параметры Описание
map.layers (layer ID) Выбирает слой по его ID внутри слоев заданной карты
Метод Параметры Описание
set.Visible (true, false) Позволяет выборочно отключать слои внутри заданной карты
Добавление объектов
Function(Map)
Класс Функции Описание
map drawing add.Object Добавляет на карту объекты описанных типов (см. типы объектов)
tools Выбирает элементы пользовательского интерфейса, активирующие режим рисования данных объектов
miniMap setVisible Отключение окна миникарты
addLayer Задание слоя для отображение в окне миникарты по его ID
coordinates setVisible Отключение координат
scaleBar setVisible Отключение масштабной шкалы
Функция Параметры Описание
addObject "LINESTIRNG" линия
"POLYGON" многоугольник
"POINT" точка
addLayer ("Layer ID") ID слоя
tools "LINESTIRNG" элемент включения режима "рисование линии"
"POLYGON" элемент включения режима "рисование многоугольника"
"POINT" элемент включения режима "рисование маркера"
setVisible (true, false) Позволяет отлючать элементы интерфейса
