Общее описание
Программный продукт GeoMixer предназначен для создания геоинформационных проектов, объединяющих в себе геоданные в растровом и векторном виде. С его помощью возможно организовывать пространственные данные в наборы слоев (карты) и управлять ими с целью предоставления и оптимизации многопользовательского доступа к ним через Интернет или локальную сеть.
Компоненты системы
Программный продукт состоит из следующих компонент:

    1. GeoMixer Admin - администрирование Базы Данных, осуществляемое через веб-интерфейс: управление слоями, картами и правами пользователей (подготовка исходных данных, занесение описания в БД, настройки стилей отображения слоев, авторизация и разграничение прав доступа пользователей для просмотра публикации и редактирования данных)

    2. MapViewer Template - шаблон веб-страницы пользователя с интерактивным окном просмотра карты и панелью управления картой, в которой отображаются список слоев и дополнительная информация

    3. MapViewer API - программный интерфейс на Java Script для встраивания карт в страницы веб-сайтов
Сведения о пространственных данных
Данный сервис использует геодезическую систему координат WGS-84. Данные записываются в равноугольную цилиндрическую проекцию Меркатора и раскладываются на сервере по слоям.

Растровые данные, подготовленные для просмотра в веб-браузере средствами MapVIewer Template или API - это слой, который хранится на сервере в виде фрагментов растровых изображений (тайлов), приведенных в одну проекцию и подготовленных для каждого из заданного диапазона уровней детализации карты.
Векторные данные не требуют предварительного трансформации в базовую проекцию и так же, как и растровые данные, организуются по слоям в СУБД. серверного приложения, управляемого через веб-интерфейс GeoMixer Admin.
Масштабная шкала
Масштабный ряд, в котором показываются слои карты соответствует заданным значениям разрешения растровых изображений. Диапазон значений может быть задан от 0 и до максимального, в зависимости от выбранного Адмимнистратором данной карты.

Системные требования
Операционная система: Windows

Web сервер: IIS.

Пакет программного обеспечения:

(Серверное) Microsoft .NET Framework 3.5. MS SQL Server -2008.

(Клиентское) Браузер Internet Explorer, Mozilla Firefox, Opera с установленным плагином Adobe Flash Player.
